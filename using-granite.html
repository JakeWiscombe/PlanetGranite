Prompting Granite

Granite is a family of large language models created by IBM. Through using Granite AI over the past few months, I have learned best practices on how to interact with Granite to get the best possible results.

Roleplay

Roleplay helps to give Granite a framework to build from by providing a blurb at the start of the prompt that asks the AI to take on the personality, characteristics, or patterns of its assigned role. For example, a specific roleplay is "You are a Unity 6 expert and a senior C# programmer." Providing the role to Granite allows it to understand the context it is working within. Once Granite knows it is making C# scripts for Unity, then it knows to use Unity using statements for the start of C# scripts.

Modes

Granite has a web version called Granite Playground. Granite Playground has three different modes: default, search, and research. These modes drastically change the result that the AI provides because of how Granite produces information differently. To demonstrate the differences, Granite was put through an experiment. In the experiment, Granite was asked the same question in different modes, "Is Unity 6 Out?". Default mode answered, "As of my latest update in April 2023, Unity 6 has not been released." This means that the data that the default mode model is trained on only covers data before April 2023. Newer software and, specifically, programming rules for Unity would not be reflected accurately in generated responses. Next, search mode responded, "Yes, Unity 6 is out." Search mode, as the name describes, searches the internet, meaning it will have access to information as recent as today. Lastly, research mode's answer was in the form of a long research paper, looking at different aspects of Unity 6, comparing the different versions and features, with the eventual answer of Unity 6 being out. Learning the differences between these three different modes helps users use the correct mode for what they are trying to accomplish.

Using Research for Context

Learning to use Granite Playground in default mode to produce scripts leads to outdated and incorrect information, due to it being trained on older data. Using the research mode gets around this outdated and incorrect information. A prompt like "You are a Unity 6 expert, research how best to implement this feature," then describing the feature, gives better script generations, although sometimes it would leave out an import line, or not connect correctly to another script due to its short context window. Research provides up-to-date information on Unity 6, allowing it to be an actual "Unity expert" and providing a large amount of context. Users can only enter eight thousand, but the research papers Granite produces can average eight thousand, three hundred and ten characters in length. These research papers produce a great chunk of context for it to understand the task. The specialized context is perfect because Granite will generate its own questions to research, and they are solid, in-depth questions. These questions create optimal context better and, in less time, than a person.
