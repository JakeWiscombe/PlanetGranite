Project Overview

The aim was to explore how IBM’s Granite LLM could be used in a space‑themed game. The brief encouraged experimentation with Granite both as a development tool and as an in‑game component.

Our team focused on Granite as a development tool - supporting idea generation, coding, debugging and problem‑solving.

The concept was inspired by space exploration, which influenced both the game and this blog. Instead of a traditional report, we designed an immersive website with a planetary navigation system (rocket animation + SVG flight paths). This reinforced the visual identity of the project and made the documentation process more engaging.

The blog documents the full journey: concept development, experimentation, proof‑of‑concept, technical challenges, feedback and final outcomes. It also reflects on how the team adapted to problems and how collaboration shaped the result. Overall, the project shows how AI tools like Granite can support creative and technical work in game development.
