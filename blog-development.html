Blog Development

Ildi's post

The blog was designed collaboratively by myself and Jake, with the aim of creating an immersive space-themed experience that reflected the atmosphere of the game. Instead of using a traditional navigation menu, we developed an interactive system where planets acted as navigation elements within a stylised universe background. This approach encouraged users to explore the website visually while maintaining a clean and modern design.

During the design and development process, I used Granite Playground and found it particularly useful for generating ideas on how to achieve specific visual styles and effects. The coding of the blog was completed collaboratively by Jake and myself, while Granite Playground and other LLM tools were used to assist with vibe coding, experimentation, troubleshooting, and visual design ideas.

We worked effectively as a team by regularly sharing project files and independently developing different areas of the website. After receiving the initial layout structure, I focused primarily on designing and developing the animated background system, which required many iterations, prompt refinements, and experimentation to achieve the final visual style.

The animated background was created using the HTML5 canvas element and the JavaScript Canvas 2D API. The system consisted of multiple procedural layers including stars, dust particles, and nebula clouds, each generated using JavaScript arrays containing properties such as position, opacity, size, and movement speed. Depth effects, gradient lighting, and additive blending techniques were used to create a more atmospheric and realistic environment.

The animation loop was controlled using requestAnimationFrame() to ensure smooth rendering and improved performance. Optimisation techniques such as particle recycling, lightweight calculations, and canvas-based rendering were implemented throughout development to maintain responsiveness across devices.

I was also responsible for designing the individual content pages of the blog. These pages retained the overall space aesthetic while simplifying the animation with static stars to improve readability and reduce visual distraction.

Blog Development

Jake's post

During the development of the Planet Granite blog, a major part of my contribution was the designing and planning of the blog, as well as focusing on debugging and refining interactive elements of the website, particularly the rocket animation that travels between the planets. The intended effect was for the rocket to follow the curved SVG path smoothly as the user scrolled through the page. However, during testing, the rocket didn’t start or end in the correct position and rotated incorrectly – breaking the immersion of the experience and made the navigation feel broke and inconsistent.

To resolve this issue, the JavaScript controlling the animation was completely reworked, alongside refining the SVG path itself to allow a smooth aesthetic when scrolling. I experimented with different methods of calculating the rocket’s position using path length and scroll percentages, eventually improving the accuracy of the movement so the rocket aligned more naturally with the curve.
